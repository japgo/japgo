#

#### 주요 내용
* Vision 검사를 통해 Display Module 및 Cover glass의 외관 불량을 자동으로 검사.
* 수동 육안 검사 대비 수율/품질 향상
* 불량 Data 축적 및 관리, 검사 인력 최소화.
* CIM Program 연동하여 공장 자동화 및 Inline 생산 최적화.
#

#### 담당 업무
* 장비 제어 관련 부가 기능 개발.
* 장비 제어 UI(C#) 개발
#

#### 주요 실적
* 삼성 디스플레이 (베트남) 납품 후 양산 진행 중.
